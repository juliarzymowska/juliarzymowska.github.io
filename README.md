# JRZ | Personal Page

My personal corner of the web. Built without frameworks, relying entirely on pure HTML, CSS, and vanilla JavaScript. 

This site serves as my portfolio, digital garden and whatever will come to my mind. It is designed to be completely tracker-free.

## 🛠️ Tech Stack

* **HTML5**
* **CSS3** 
* **Vanilla JavaScript** 
* **Zero Dependencies.** No React, no Tailwind, no npm packages.

## 🚀 Local Deployment

To run this site locally:

1. Clone the repository:
   ```bash
   git clone [https://github.com/juliarzymowska/juliarzymowska.github.io.git](https://github.com/juliarzymowska/juliarzymowska.github.io.git)
   ```
2. Navigate to the directory:
    ```bash
    cd juliarzymowska.github.io
    ```
3. Open `index.html` in any browser!
